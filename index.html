Cáptiulo 1 - Estrutura

Resumo Estrutura

  • Páginas HTML são documentos de texto.
  • A HTML usa tags (caracteres entre colchetes angulares para dar às informações entre essas tags um significado especial.
  • Tags muitas vezes são chamadas de elementos.
  • Tags geralmente vêm em pares. A tag de abertura indica o início de uma parte do conteúdo; a tag de fechamento indica o fim.
